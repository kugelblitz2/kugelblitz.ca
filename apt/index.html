kugelblitz's Debian Repository

This repository contains some useful packages not found in the official debian repositories.

How to use

First, add my key:

curl https://kugelblitz.ca/apt/apt.key | sudo apt-key add -

Add the following to /etc/apt/sources.list :

deb [arch=amd64] https://kugelblitz.ca/apt/repo buster main
